JD Stairs - Website Planning Document

Informational Website Site Plan

Site Name

Site Name: JD Stairs

I chose this name because it is the name of the business that currently exists in Las Vegas. It is already known locally in Las Vegas for custom stair craftmanship and by using it's name helps with recognition and credibility of the company.

Optional name: jdstairsinfo.com

Site Purpose

  • What types of custom work does JD Stairs specialize in?
  • How can I contact JD Stairs to request a quote for my home or business?
  • Can I see examples of previous stair projects before deciding?

Color Scheme

  • Dark Charcoal (#2c2c2c): Used for heading, header background, and primary text emphasis.
  • Wood Brown Accent (#8b5a2b): Used for section borders and accent details to reflect wood craftmanship
  • Light Gray (#f7f7f7): Used as the background color.

Typography

  • Lora: This is going to be my font for my headings
  • Montserrat: This will be the font for the body text
